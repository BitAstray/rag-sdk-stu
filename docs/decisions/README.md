## ADR 格式

每个决策记录包含：

- **状态：** 已采纳 / 已废弃 / 已取代
- **背景：** 为什么需要做这个决策
- **决策：** 具体做了什么选择
- **原因：** 为什么选择这个方案
- **后果：** 这个决策带来的影响
- **引用：** 相关文档和资源
